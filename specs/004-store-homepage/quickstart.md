# Quickstart: Store Homepage Feature

## Prerequisites
- Node.js 18+
- Frontend dependencies installed (`npm install` in `frontend/`)

## Running the Feature

1. Navigate to frontend:
   ```bash
   cd frontend
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open browser to `http://localhost:5173` (or port shown in terminal).

4. You should see:
   - **Navbar**: Top navigation with "Categories" dropdown.
   - **Hero/Promotion Block**: Displaying "滿 10,000 打 95 折" info.
   - **Featured Products**: A grid of highlighted products.

## Testing

Run component tests:
```bash
npm run test:unit
```
